# Bienes Raices Page - HTML & Tailwind CSS

Bienes Raices page using Tailwind CSS

## Setting up

Install dependencies:

```bash
yarn install
# or
npm install
```

Build the CSS

```bash
yarn build:css
# or
npm run build:css
```

## This project use

All the important library's that the project use.

- Tailwind CSS
